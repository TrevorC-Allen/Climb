# Climb

一个从零开始的攀岩训练记录与动作分析原型。

## 当前功能

- 训练记录：岩馆、日期、抱石/顶绳/先锋、V Scale / YDS 难度、结果、尝试次数、体感强度、风格标签和备注。
- 成绩看板：训练日、完成率、稳定抱石、稳定难度、Grade Pyramid、最近记录和提升轨迹。
- 训练建议：根据完成率、项目线、风格短板、训练负荷和动作片段生成建议。
- 动作分析：摄像头或导入视频会尝试加载 MediaPipe Pose Landmarker，识别骨架后检测锁臂、重心偏移、三点支撑和开门风险；模型不可用时会回到规则演示模式。
- 野攀路书：通过 OpenBeta GraphQL API 搜索野攀区域、预览线路、收藏区域，并保留 OpenBeta / theCrag 站外入口。
- 本地账号：支持在同一设备上创建多个本地账号，训练记录、动作片段和路书收藏按账号隔离。
- 本地数据：使用浏览器 localStorage 保存，支持按当前账号 JSON 导入导出。
- 产品路线图：内置短期、中期、长期路线图，并记录 Apple Fitness、Garmin Connect、Strava、Nike Run Club 等产品参照。

## 运行

直接打开 `index.html` 即可使用。

如果需要摄像头预览，建议通过本地服务器打开：

```bash
python3 -m http.server 5173
```

然后访问：

```text
http://localhost:5173
```

## 手机独立运行

这个项目已经是 PWA。部署到 HTTPS 后，在手机浏览器打开一次并添加到主屏幕，之后离开电脑也能打开。

推荐最省事路线：

1. 把仓库发布到 GitHub Pages、Netlify、Vercel 或 Cloudflare Pages。
2. 用手机打开发布后的 HTTPS 地址。
3. iPhone Safari：分享 -> 添加到主屏幕。
4. Android Chrome：菜单 -> 添加到主屏幕 / 安装应用。

数据保存在手机浏览器本地；不同设备之间可以用 JSON 导出/导入迁移。

## 账号系统

当前是短期版本：本地账号，不需要服务器，不包含密码和云同步。

- 第一次打开会自动把旧数据迁移到“我的账号”。
- 可以在右上角账号按钮里创建、切换和删除本地账号。
- 每个账号都有独立的训练记录、动作片段和路书收藏。
- 删除账号会删除该账号在当前设备里的本地数据。
- 中期版本会接入云端登录和多设备同步。

## 路线图

短期：打磨训练日志、训练 session、岩馆分店、视频动作时间轴和周报。

中期：加入训练计划、负荷与恢复、风格画像和项目线管理。

长期：支持多设备同步、个性化动作模型、岩馆/野攀路书生态和月度教练报告。

产品参照：

- Apple Fitness：摘要、趋势、奖章和分享。
- Garmin Connect：训练状态、负荷、恢复和洞察。
- Strava：目标、挑战、分段对比和社区动力。
- Nike Run Club：训练计划和教练化引导。
